Cara masukkin file ke git repository

Caranya adalah config dulu dulu ke git dengan cara ketikkan :

git config --global user.email "your@example.com"
git config --global user.name "Your name"



kemudian check dengan git config --list apakah sudah masuk atau belum

yang kedua:

tulis kan code di bawah untuk menambahkan file .git yang akan memantau semua perubahan yang terjadi pada direktori tersebut

git init



yang ketiga

Buat dulu repository di GitHub lalu sambungin repository yang akan kamu buat di Git dengan GitHub pakai kode :

git remote add nama_remote <https:ww.web-github-repository-kamu.com ≷



cara ngecek remote yang sudah kamu bikin pakai kode:

git remote



yang keempat:

masukkan file yang mau di masukin pake kode:

git add nama_file

nanti kalau kamu mengguknakan visual studio code terdapat kode di samping nama file:

Arti kode pada git
Kode Arti
U Untracked
A Index Added
M Modified


yang kelima:

masukkan kode berikut untuk memasukkan perubahan pada file:

git commit -m "status file"



yang keenam:

masukkan file ke github dengan perintah push :

git push -u nama_remote nama_branch



Usual Command Git

Configuring Git

git config --global user.name "Your Name"
Set the name that will be attached to your commits
git config --global user.email "Youremail@gmail.com"
Set the email that will be attached to your commits
git config -- --global core.editor "code --wait"
Set the default editor used by Git
